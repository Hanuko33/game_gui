## Keys

* ESC - menu: exit, load, save, help menu
* m   - menu: energy
* arrows, w, a, s, d - moves
* r - switch running
* e, ENTER - interact / pickup item
* F5 - autoexplorer
* F11 - stop automaticaly resizing (may break some things)
